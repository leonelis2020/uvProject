# ─── Streamlit 버전 핀 — 마우스 freedraw 마스킹 호환을 위해 1.41.x 고정 ─────
#
# streamlit-drawable-canvas==0.9.3 는 `streamlit.elements.image.image_to_url`
# 의 옛 시그니처(width:int)에 의존한다. Streamlit 1.42 부근에서 해당 API 가
# `streamlit.elements.lib.image_utils.image_to_url(image, layout_config, ...)`
# 로 옮겨지면서 시그니처가 바뀌어 0.9.3 이 깨진다. 호환 monkey-patch 도
# layout_config 객체 형태가 내부적으로 변하면서 실패했다(PR #6 회고).
#
# 따라서 1.36 ≤ Streamlit < 1.42 가 안전 구간이며, 그 안에서 가장 안정된
# 마지막 패치 릴리스가 1.41.1 이다. 본 프로젝트가 쓰는 모든 기능
# (@st.dialog 1.35+, st.connection 1.28+, st.container(border=) 1.29+) 이
# 1.41 에 포함되어 있어 기능 손실은 없다.
#
# 이미 더 높은 버전이 설치된 환경에서는 pip 가 기본적으로 다운그레이드 안 하므로
# 강제로 재설치하라:
#     pip install --upgrade --force-reinstall -r requirements.txt
